◆ Increment_Passed_Cycles()

virtual void zero_mate::IExternal_Peripheral::Increment_Passed_Cycles ( [[maybe_unused] ] std::uint32_t  count)
inlinevirtual

Notifies the peripheral about how many CPU cycles have passed by.

The CPU core calls this function after every instruction, so the peripheral can update itself.

Note
In reality, external peripherals do not know of CPU cycles - simplification (e.g. UART)
Parameters
countNumber of CPU cycles

◆ Set_ImGui_Context()

virtual void zero_mate::IExternal_Peripheral::Set_ImGui_Context ( [[maybe_unused] ] void *  context)
inlinevirtual

Sets ImGuiContext which is used when the peripheral decides to render itself.

The current ImGuiContext needs to be set before the peripheral's GUI is rendered.

Note
The ImGuiContext is passed as a void pointer to avoid compile-time dependency on the ImGUI library which is used as a GUI library in the main executable (emulator).
Parameters
contextPointer to an ImGuiContext
