void zero_mate::arm1176jzf_s::CCPU_Context::Init_Registers ( )
inlineprivate

Initializes (resets) all CPU registers.

Registers r0-r15 in all CPU modes are set to 0. The CPSR registers of all modes hold only info about their corresponding mode (least significant 5 bits). The SPSR registers are cleared out to 0.

bool zero_mate::arm1176jzf_s::CCPU_Context::Invalid_Change_Of_Control_Bits ( std::uint32_t  new_cpsr)
privatenoexcept

Checks if there is an invalid attempt to change the control bits of the CPSR register.

The control bits of the CPSR register can be changed only from a privileged mode. If the CPU is in the User mode, it is not allowed the change the least significant byte of CPSR (control bits).

Parameters
new_cpsrNew value of the CPSR register
Returns
true, if an invalid attempt takes place. false, otherwise.
