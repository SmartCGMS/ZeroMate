Detailed Description

This class represents an extend instruction.

◆ NCategory_Mask

enum zero_mate::arm1176jzf_s::isa::CExtend::NCategory_Mask
strongprivate

This enumeration is used to determine the type of the extend instruction.

It is a helper technique to figure out the type using different bit masks (bits 22, 21, and 20). Category_1_1 means the first type in category (1).

enum zero_mate::arm1176jzf_s::isa::CExtend::NType : std::uint32_t

This enumeration represents different types of the extend instruction.

Enumerator
SXTAB16 

Sign extend two Bytes with Add, to extend two 8-bit values to two 16-bit values.

UXTAB16 

Zero extend two Bytes and Add.

SXTB16 

Sign extend two bytes.

UXTB16 

Zero extend two Bytes.

SXTAB 

Sign extend Byte with Add, to extend an 8-bit value to a 32-bit value.

UXTAB 

Zero extend Byte and Add.

SXTB 

Sign extend Byte, to extend an 8-bit value to a 32-bit value.

UXTB 

Zero extend Byte.

SXTAH 

Sign extend Halfword with Add, to extend a 16-bit value to a 32-bit value.

UXTAH 

Zero extend Halfword and Add.

SXTH 

Sign extend Halfword.

UXTH 

Zero extend Halfword.
