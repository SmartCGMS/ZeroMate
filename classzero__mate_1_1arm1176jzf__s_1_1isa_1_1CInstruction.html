enum zero_mate::arm1176jzf_s::isa::CInstruction::NCondition : std::uint32_t
strong

This enumeration defines different condition flags that can set in an instruction.

Whether a condition is going to be executed is determined by these flags. Check out https://developer.arm.com/documentation/dui0801/g/Condition-Codes/Condition-code-suffixes.

Enumerator
EQ 

Z set; equal.

NE 

Z clear; not equal.

HS 

C set; unsigned higher or same.

LO 

C clear; unsigned lower.

MI 

N set; negative.

PL 

N clear; positive or zero.

VS 

V set; overflow.

VC 

V clear; no overflow.

HI 

C set and Z clear; unsigned higher.

LS 

C clear or Z set; unsigned lower or same.

GE 

N equals V; greater or equal.

LT 

N not equal to V; less than.

GT 

Z clear AND (N equals V); greater than.

LE 

Z set OR (N not equal to V); less than or equal.

AL 

always

Unconditioned 

always (condition is not applied)
