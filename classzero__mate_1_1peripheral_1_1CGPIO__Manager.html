Enables/disables an interrupt for all pins within a given register.

If a bit is set to a 0, the interrupt type will be disabled. If a bit set to a 1, the interrupt type will be enabled.

Parameters
reg_idxIndex of the register (GPREN, GPHEN, GPLEN, or GPFEN)
last_regIndication of whether it is the last register (not all bits are being used)
