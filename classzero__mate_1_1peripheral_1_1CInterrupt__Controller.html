enum zero_mate::peripheral::CInterrupt_Controller::NIRQ_Source
strong

This enumeration defines different IRQ source that can trigger an interrupt.

Each value represents a bit position in the corresponding enable/disable register. If the position is greater than 31, register (2) is used.

bool zero_mate::peripheral::CInterrupt_Controller::Has_Pending_Interrupt ( ) const
noexcept

Returns information about whether there is a pending IRQ.

For this function to return true, all interrupts must be globally enabled in the CPSR register. Additionally, there must be a basic IRQ pending or an IRQ pending (see Has_Pending_IRQ, and Has_Pending_Basic_IRQ).

Returns
true, if there is an pending interrupt. false, otherwise.
