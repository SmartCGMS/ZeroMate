Detailed Description

template<typename Type>
class zero_mate::utils::CSingleton< Type >

A generic class that allows for treating other classes as singletons.

If a class is meant to be treated as a singleton using CSingleton, it needs to implement and empty constructor, so CSingleton can instantiate it. CSingleton works as a proxy that provides access to the instance of the class.
