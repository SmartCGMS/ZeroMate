Creates an instance of an external peripheral.

This function is called outside the shared library. It is called by the emulator when it attempts to attach the peripheral to the GPIO pins.

Parameters
peripheralPointer to a pointer to an external peripheral (the actual pointer is in the emulator)
nameUnique name of the peripheral (e.g. "My button")
gpio_pinsCollection of GPIO pins the peripherals is hooked up to
pin_countNumber of GPIO pins used by the peripheral
set_pinReference to a function the peripheral can use to set GPIO pins
read_pinReference to a function the peripheral can use to read values of GPIO pins
logging_systemPointer to the logging system that is used throughout the project
Returns
0, if all goes well and the peripheral is created successfully. 1, otherwise.
