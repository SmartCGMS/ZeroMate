Multiples two 32-bit values (the result it treated as a 32-value as well).

If the A bit is set in the isa::CMultiply instruction. It adds the reg_rn_val value to the result of reg_rm_val * reg_rs_val.

Multiples two 32-bit values (the result is a 64-bit value split up into two 32-bit values).

if the A bit is set in the isa::CMultiply_Long instruction, ((reg_rd_hi << 32U) | reg_rd_lo) is added to the final result of the multiplication (reg_rm_val * reg_rs_val). The result is treated as a 64-bit value split up into two 32-bit values.

Parameters
instructionMultiply long instruction
reg_rm_valFirst operand
reg_rs_valSecond operand
reg_rd_loLower 32 bits of the accumulate value
reg_rd_hiHigher 32 bits of the accumulate value
Returns
reg_rm_val * reg_rs_val is the A bit is not set. Otherwise, it returns (reg_rm_val * reg_rs_val) + ((reg_rd_hi << 32U) | reg_rd_lo).
